Das Right

The word “rights” is the most prominent example of language abuse for politics sake. It is short and not some yet another pseudo scientific “-ism”, “-fobia”. So it looks more natural on the outset; It has “right” in it :) You can create new “rights” every day by adding different perfixes; You can use it to extract something from somebody; Since it is vague term it can have broad uses especially when it comes to general public; It also informs users on how to act/behave when demanding something from others: righteously; ...
